Ntnu-class-signature
===
### 目的
* 使用教室名稱，達到簽到效果。

### 開發環境 
* Xcode 11.3.1
* MacOS 10.15.2
* IOS 13 or late

### Library
* pod

```
  pod 'Alamofire', '~> 5.0.0-rc.3'
  pod 'SwiftyJSON'
  pod 'RxSwift', '~> 5'
  pod 'RxCocoa', '~> 5'
  pod 'RealmSwift'
```
### Use Guide
* Open the xcode workspace from: `Ntnu-class-signature/Ntnu-class-signature/Ntnu-class-signature.xcworkspace`
* build and run
